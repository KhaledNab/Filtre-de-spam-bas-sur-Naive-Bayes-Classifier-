    Filtre de spam basé sur "Naive Bayes Classifier"

* Ce notebook a comme objectif d’appliquer le principe de classification des courriels basé sur "Naive Bayes Classifier" pour filtrer les spams. 

* En principe ce dernier est un algorithme de Machine Learning consiste à calculer le nombre d’occurrence des mots dans chaque mail et par la suite calculer la probabilité conditionnelle d'apparaitre dans un spam. 

* Python dispose de libraires prédéfinies dédiées à ces calculs. Cet environnement est livré avec de nombreuses bibliothèques d'analyse utiles installées.
